    base¶

    Base classes for prompt decorators.

    This module provides the base classes and utilities for creating and using prompt decorators.

    Classes¶

    BaseDecorator¶

    Base class for all prompt decorators.

    This class defines the common interface and behavior for all decorators. Subclasses should implement the apply_to_prompt and transform_response methods.
